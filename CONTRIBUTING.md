
# 貢獻指南 / Contributing Guide

感謝您對 Mirrorwalker-AI 的興趣！本專案歡迎任何形式的貢獻，包括錯誤回報、功能建議、樣本改進、文件完善等。

## 如何參與

1. **回報問題 / Report Issues**  
   - 在 GitHub 上建立 Issue，描述您遇到的問題或建議。

2. **提交 Pull Request**  
   - Fork 本倉庫，建立您的功能分支，提交 Pull Request 前請確保：
     - 符合編碼風格。
     - 測試通過（如有）。
     - 說明您改動的目的與內容。

3. **樣本或文檔貢獻**  
   - 提交新的裂解樣本、感染曲線或改進文檔，請遵循現有格式。
